# list

List available and installed components.

## Usage

```bash
aneka-ui list
```

## Examples

```bash
# List all components
aneka-ui list

# List only installed
aneka-ui list --installed

# List only available
aneka-ui list --available
```

## Options

| Option | Description |
|--------|-------------|
| `--installed` | Show only installed components |
| `--available` | Show only available components |
