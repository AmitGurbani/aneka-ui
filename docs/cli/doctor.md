# doctor

Diagnose and fix setup issues.

## Usage

```bash
aneka-ui doctor
```

## Examples

```bash
# Check for issues
aneka-ui doctor

# Auto-fix issues
aneka-ui doctor --fix
```

## Options

| Option | Description |
|--------|-------------|
| `--fix` | Automatically fix detected issues |

## What It Checks

- Configuration file validity
- Required dependencies
- Path aliases
- Tailwind setup
- TypeScript configuration
